# Cats vs Dogs Classification using SVM

## Overview

This project classifies images of cats and dogs using Support Vector Machine (SVM) and Machine Learning.

## Technologies Used

- Python
- NumPy
- OpenCV
- Matplotlib
- Scikit-learn

## Features

- Image preprocessing
- Grayscale image conversion
- Image resizing
- SVM classification model
- Prediction visualization
- Accuracy evaluation

## Dataset

Dogs vs Cats Dataset from Kaggle

## Output

The model predicts whether the image is:

- Cat
- Dog
